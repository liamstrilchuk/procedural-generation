Procedural Generation

Liam Strilchuk

Have you ever played a game like Minecraft or Terraria? If so, you'll have noticed a fundamental difference between them and other games. They have a world that is nearly infinite, and can be explored for as long as you want. The way they do this is using something called "procedural terrain generation".

Procedural terrain generation is the creation of realistic terrain using a computer, instead of making it manually. While it may take weeks or more for a designer to create a video game level, it can be done in seconds using computers.

This procedure has many benefits over traditional creation of levels. It can be implemented much more quickly, and does not require skilled designers. Some games, such as Grand Theft Auto, have dozens of developers and have taken years to make. It also gives more replayability—the world is different on every playthrough, and the user will not get bored as quickly. Creating a completely different type of world is also trivial, as changing a few settings can drastically change the output. Finally, the world is nearly infinite in size. Users can explore them for as long as they want, instead of being limited to a few levels.

However, there are also some drawbacks. First, procedurally generated worlds can't be anywhere near as detailed as one created by a human. Levels can also be repetitive—while the worlds are not the same, the content is similar. Bugs and other types of errors are more common, as not every level can be tested. Finally, it is much harder to include a realistic storyline in the game if the maps are not consistent, and thus most procedurally generated games do not have an in-depth story.

There are other algorithms, similar to perlin noise, that are also used. One of these is called "simplex noise". Simplex noise generates data in a similar fashion to perlin noise, but much faster. However, its results are "rougher", and harder to use.

Other uses of procedural generation

There are many other uses for procedural generation. Creating terrain for games is the most well known use, but it is used in many other fields of computer science. In the last few years, new programs have been developed that can create music without any human intervention. The results of these programs can't be told apart by humans—computers can now create songs that sound exactly the same as ones created by people.

Movies and television shows often use procedural generation for landscapes. For example, The Mandalorian's scenes were made using a game engine, and displayed on a screen behind the actors.

In conclusion, procedural generation is an extremely powerful tool that can be used by game developers. It can save time, and creates a world that can be explored almost infinitely.
